# 示例：创建一个事件世界

这个文档给出一个最小使用示例，方便第一次运行 AionCausa。

## 示例输入

```text
如果商鞅变法成功后没有被车裂，秦国会如何发展？
```

## 推荐流程

1. 在首页输入中心事件。
2. 点击 `分析可创建性`。
3. 等待模型判断世界是否可创建。
4. 如果预检通过，点击 `生成沙盘`。
5. 进入世界现场后，观察散点舞台上的人物。
6. 点击人物点，查看人物当前意图和行动记录。
7. 点击 `推进下一幕`，让世界继续发展。
8. 推进若干幕后，点击 `总结世界`，查看当前世界整体发生了什么。

## 预期效果

模型会尝试补全：

- 商鞅为什么可能没有被杀。
- 秦孝公死后，新君和旧贵族如何反应。
- 商鞅如何保住新法与自身位置。
- 嬴驷、甘龙、公子虔、魏惠王等人物可能采取的行动。
- 秦国权力结构和扩张路径如何变化。

实际生成结果会受模型能力、输入清晰度和参数配置影响。

## 一个不适合创建世界的例子

```text
如果张三没有离开，会怎么样？
```

这个输入缺少人物出处、事件背景、改变点和世界约束。系统应该拒绝创建，或者提示用户补充信息。
